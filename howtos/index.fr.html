Raccourcis clavier

Utiliser le clavier permet d’accélérer votre travail avec Geco. La touche Mod ci-dessous fait référence à la touche d’accès au menu spécifique à votre plate-forme (alt sous Windows/Linux, cmd sous Mac).

Quand vous passez d’un onglet à l’autre, Geco met le focus sur le widget le plus important dans l’onglet. Par exemple, vous avez accès directement à la table dans l’onglet coureurs. Dans l’onglet résultats, le bouton Actualiser est sélectionné : pressez la touche espace pour l’activer.

Raccourcis d’accès aux onglets

  • Mod+nombre (1 à 9) : focus sur l’onglet correspondant, de gauche (1) à droite. Par example, Mod+1 met le focus sur l’onglet Étape, Mod+4 sur l’onglet résultats…
  • Mod+0 : focus sur le bouton Lecture dans la barre d’outils (espace pour activer/désactiver la lecture)

Raccourcis des tables de coureurs (onglet coureurs et archive)

  • Mod+F : focus sur le champ de recherche
  • Mod+Shift+F : annuler le filtre de recherche
  • Mod+I : insérer une nouvelle entrée (onglet coureurs)/insérer le coureur sélectionné (vue archive)

Raccourcis dans l’onglet coureurs

  • Mod+D : activer la vue Données du coureur
  • Mod+T : activer la vue Trace du coureur
  • Mod+S : activer la vue Statistiques de course
  • Mod+R : vérifier le coureur sélectionné (raccourci pour le bouton Vérifier)
  • Mod+P : imprimer les temps intermédiaires du coureur sélectionné (raccourci pour le bouton Imprimer Splits)
  • utilisez les touches tab, flèches, espace, échap pour naviguer dans la table et démarrer/arrêter l’édition des cellules de la table
  • Mod+L: activer/désactiver le mode Live
