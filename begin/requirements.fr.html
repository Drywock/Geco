Pré-requis

Pour faire tourner Geco, vous avez besoin :

  • d’un JRE récent (Java Runtime Environment, version 6+)
  • d’une archive Geco qui contient l’application (fichier jar), les fichiers modèles pour une course, la documentation, les bibliothèques natives de communication série
  • des pilotes SPORTident
    • Windows : voir le site SPORTident
    • Linux : les noyaux récents reconnaissent la puce utilisée par les boîtiers maîtres, plug’n’play !
    • Mac OS X : il est possible d’installer un pilote pour reconnaître le boîtier maître sous Mac. Voir la FAQ.

Pour lancer Geco, exécuter le fichier geco.jar. Il devrait se lancer directement si votre JRE est bien configuré.

Bibliothèque RXTX

Dans certains cas, vous devrez installer et compiler les pilotes RXTX (pilotes série pour plateforme Java). Contactez-moi si besoin.
